#### Docker

A Docker image of this project is available for
    use.

  - [timtrice/distill](https://cloud.docker.com/repository/docker/timtrice/distill)

##### Run

    docker run \
      -dti \
      -e DISABLE_AUTH=true \
      -p 8787:8787 \
      --name distill \
      timtrice/distill:release

##### Shell

    docker exec -ti distill /bin/bash
